# Simple-calculator
A simple calculator in python. Using **tkinter** for the UI.
- First using python's built-in **eval** - simplest method but not recommended.
- Implementation using a **custom function** that interprets the string from the input field. Handles the risk of using eval but fails with numbers with multiple digits. Split() works for such numbers if there are spaces between the numbers and symbols.
- Implementation using **regex**.
- Records calculations **history**.
